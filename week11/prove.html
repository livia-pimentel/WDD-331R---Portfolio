Week 11 - Prove Debugging and Optimization

Webpage analyzed: https://www.stumbleguys.com/pt-BR
This is an online game used a lot by children in my region (Bahia - Brazil). The analysis was carried out on the basis of the mobile.

Tool used: The Lighthouse

The Lighthouse Report: Mobile

Recommendations to improve website performance

  • Replace PNG or JPEG format images with formats such as WebP or AVIF.
  • Provide images with adequate sizes.
  • Reduce unused CSS and JavaScript.
  • Add Accessible Names to Button Tags.
  • Increase the contrast ratio between colors.

These simple changes will improve the page load time and increase the accessibility of the site, there are free tools to convert images from PNG or JPG to WebP or AVIF one of them is Convertio, the same goes for resizing the images a free tool is iloveimg.
It is possible to identify unused files with dev tools by typing the coverage command in the console, and to test the color contrast there are several tools for this, one of which is Webaim.
